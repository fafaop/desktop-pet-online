# Error Code Definition

## Common

|Code|Description|
|-|-|
|0|Success|
|1000|Invalid parameter|
|1001|Unsupported request|

## Authentication

|Code|Description|
|-|-|
|2000|Authentication failed|
|2001|Token expired|
|2002|User not found|

## Message

|Code|Description|
|-|-|
|3000|Message send failed|
|3001|Receiver offline|

## Server

|Code|Description|
|-|-|
|5000|Internal server error|
|5001|Service unavailable|
