# Desktop Pet Online

## 项目简介

桌面宠物联机版是一款基于 Windows 平台的多人互动桌面应用。

用户可以在桌面养成虚拟宠物，并通过网络实现好友聊天、宠物状态同步、互动玩法等功能。

## 项目目标

- Windows 桌面宠物体验
- 用户之间实时通信
- 支持单聊、群聊
- 宠物状态同步
- 可扩展客户端/服务端架构

## 技术规划

### Client

- Windows Desktop Application
- UI Framework
- 本地数据管理
- 网络通信模块

### Server

- 用户管理
- 身份认证
- 消息服务
- 宠物同步服务

### Common

- 通信协议
- 数据模型
- 错误码定义
