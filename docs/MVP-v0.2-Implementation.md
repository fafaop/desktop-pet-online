# MVP v0.2 Implementation Plan

## 当前基础

项目已经具备：

- Electron 客户端基础结构
- WebSocket 服务端
- 用户管理
- 房间管理
- 宠物状态模型

## v0.2目标

实现双客户端真实联机体验。

## 服务端增强

### Session

- 用户唯一 ID
- 连接状态维护
- 心跳检测
- 断线清理

### Room

- 创建房间
- 加入房间
- 房间广播
- 房间成员同步

### Sync

统一同步消息：

- CHAT
- USER_SYNC
- PET_UPDATE
- HEARTBEAT

## 客户端增强

- WebSocket 自动连接
- 消息接收分发
- 聊天窗口
- 在线玩家列表
- 宠物状态展示

## 验收

启动一个服务端，打开两个客户端：

1. 两个用户成功连接
2. 用户昵称同步
3. 房间消息互通
4. 宠物状态变化同步
